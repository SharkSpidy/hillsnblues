# 🌿Hills And Blues — Homestay Website

A **premium, serene single-page website** for a private cottage homestay in **Peerumedu, Kerala**, built using **React + TypeScript + Vite**.

Designed with a focus on **nature, minimalism, and immersive user experience**, inspired by luxury resort websites.

---

## ✨ Features

### 🌄 Landing Page

* Full-screen **hero section with cinematic visuals**
* Smooth scrolling and elegant transitions
* Sections:

  * About / Story
  * Cottage Details (Single Cottage Focus)
  * Experiences
  * Gallery
  * Contact Form
* Responsive design (mobile-first)

---

## 🛠 Tech Stack

* ⚛️ React (with TypeScript)
* ⚡ Vite (fast build tool)
* 🎨 Tailwind CSS (utility-first styling)
* 🎯 Lucide Icons
* 🌐 React Router DOM

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/SharkSpidy/hillsnblues.git
cd hillsnblues
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run development server

```bash
npm run dev
```

App will run on:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview build:

```bash
npm run preview
```
